1//!# Serde YML (a fork of Serde YAML)
2//!
3//![![Made With Love][made-with-rust]][11] [![Crates.io][crates-badge]][07] [![lib.rs][libs-badge]][12] [![Docs.rs][docs-badge]][08] [![Codecov][codecov-badge]][09] [![Build Status][build-badge]][10] [![GitHub][github-badge]][06]
4//!
5//![Serde YML][00] is a Rust library for using the [Serde][01] serialization framework with data in [YAML][05] file format.
6//!
7//! ## Features
8//!
9//! - Serialization and deserialization of Rust data structures to/from YAML format
10//! - Support for custom structs and enums using Serde's derive macros
11//! - Handling of YAML's `!tag` syntax for representing enum variants
12//! - Direct access to YAML values through the `Value` type and related types like `Mapping` and `Sequence`
13//! - Comprehensive error handling with `Error`, `Location`, and `Result` types
14//! - Serialization to YAML using `to_string` and `to_writer` functions
15//! - Deserialization from YAML using `from_str`, `from_slice`, and `from_reader` functions
16//! - Customizable serialization and deserialization behavior using Serde's `#[serde(with = ...)]` attribute
17//! - Support for serializing/deserializing enums using a YAML map with a single key-value pair through the `singleton_map` module
18//! - Recursive application of `singleton_map` serialization/deserialization to all enums within a data structure using the `singleton_map_recursive` module
19//! - Serialization and deserialization of optional enum fields using the `singleton_map_optional` module
20//! - Handling of nested enum structures with optional inner enums using the `singleton_map_recursive` module
21//! - Customization of serialization and deserialization logic for enums using the `singleton_map_with` module and custom helper functions
22//!
23//!## Installation
24//!
25//!Add this to your `Cargo.toml`:
26//!
27//!```toml
28//![dependencies]
29//!serde = "1.0"
30//!serde_yml = "0.0.12"
31//!```
32//!
33//!## Usage
34//!
35//!Here's a quick example on how to use Serde YML to serialize and deserialize a struct to and from YAML:
36//!
37//!```rust
38//!use serde::{Serialize, Deserialize};
39//!
40//!#[derive(Debug, PartialEq, Serialize, Deserialize)]
41//!struct Point {
42//!    x: f64,
43//!    y: f64,
44//!}
45//!
46//!fn main() -> Result<(), serde_yml::Error> {
47//!    let point = Point { x: 1.0, y: 2.0 };
48//!
49//!    // Serialize to YAML
50//!    let yaml = serde_yml::to_string(&point)?;
51//!    assert_eq!(yaml, "x: 1.0\n'y': 2.0\n");
52//!
53//!    // Deserialize from YAML
54//!    let deserialized_point: Point = serde_yml::from_str(&yaml)?;
55//!    assert_eq!(point, deserialized_point);
56//!
57//!    Ok(())
58//!}
59//!```
